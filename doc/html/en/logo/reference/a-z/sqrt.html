SQRT

Return the square root of the given argument.

Synopsis

sqrt <value>

Arguments

<value> - the value to find the square root of

    Returns

    Returns the square root of <value>.

    Example

              make "foo 9
          make "bar sqrt :foo    ; foo == 3

    See Also

    exp pow sqr
