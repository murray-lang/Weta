AIN

BeforeAfter
  
uint8 port#int16 ADC value

Table 1. VM Stack


Remove the uint8 port number from the top of the stack, and replace it with the int16 value read from that analog input port.

Synopsis

port#

ain

Arguments

<port#> - the required port number (uint8)

    Returns

    Returns the int16 value from the specified ADC port.

    Example

            uint8 3         ; Select port 3
        ain             ; Get analog input 3
        itostr          ; Replace the int16 with its string representation
        strsend         ; Send the string to the default output stream
