FOREVER

Execute statements in a block forever. This is exactly equivalent to loop

Synopsis

forever [<statements>*]

or...

loop [<statements>*]

Where

<statements> - zero or more program statements

    Example

                  ; Flash the user LED approximately every second
          forever
          [
              ledon
              wait 1
              ledoff
              wait 9
          ]

    See Also

    do.while for foreach if ifelse repeat for
