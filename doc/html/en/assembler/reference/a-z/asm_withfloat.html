WITHFLOAT

Abstract

Selects float (32 bit) as the type to be used for any subsequent stack operations.

Synopsis

withfloat

Arguments

Nil.

Returns

Nil. (The instruction sets a register that is internal to the virtual machine.)

Example

            .global _start
            .data
        radius:	       .single  0
        circumference: .single  0
            .text
            ...
               ; Calculate the circumference for the given radius
            withfloat
            global   radius
            get
            float    3.141592654
            float    2.0
            mul
            mul
            global   circumference
            set
            ...
            exit
            .end
