FOREVER

Abstract

Repeat the block of instructions forever.

Synopsis

block

...

eob

forever

Arguments

Nil

Example

            ; pulsar
        block
        ledon
        int16  100     ; Argument to wait (1 tenth second)
        wait
        ledoff
        int16  900     ; Argument to wait (9 tenths second)
        wait
        eob
        forever

See Also
