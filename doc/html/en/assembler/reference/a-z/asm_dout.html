DOUT

VM Stack
BeforeAfter
  
uint8 port#
uint8 value

Abstract

Remove the uint8 port number and uint8 value from the top of the stack, and send the value to the analog output port identified by port#.

Synopsis

value

uint8    port#

dout

Arguments

<value> - value to output (uint8)

<port#> - the required port number (uint8)

    Returns

    Nothing is returned to the stack.

    Example

                    ; digital output 2 follows digital input 1
            block         ; forever block
            uint8   1
            din
            uint8   2
            dout
            uint16  100   ; Wait 100ms
            wait
            eob
            forever
