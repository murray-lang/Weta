MATH.ATAN2

VM Stack
BeforeAfter
  
y-coordinate
x-coordinateangle (radians)

Abstract

Pop a y-coordinate from the top of the stack, then pop an x-coordinate. Push the angle (in radians) between the positive x-axis and the point given by coordinates x and y.

Both x and y can be positive or negative, and therefore identify a point in any quadrant.

The data type expected by math.atan2 is determined by the last with<type> instruction. The valid options for math.atan2 are: withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling math.atan2 is to use one of the prefixed versions of the the instruction, namely:

math.fatan2 or math.datan2

The assembler will expand these to a corresponding with<type> instruction followed by math.atan2.

Synopsis

x-coordinate

y-coordinate

math.atan2

Arguments

<x-coordinate> - the x-coordinate of a point in 2D space

<y-coordinate> - the y-coordinate of a point in 2D space

    Note

    This synopsis shows immediate values being explicitly placed onto the stack. However, the values could be the result peripheral I/O, function calls or any number of mathematical expressions. Or it could be an arbitrary combination of all of these.

    Returns

    Returns the angle, in radians, between the positive x-axis and the point given by coordinates x and y.
