WITHBOOL

Abstract

Selects bool as the type to be used for any subsequent stack operations.

Synopsis

withbool

Arguments

Nil.

Returns

Nil. (The instruction sets a register that is internal to the virtual machine.)

Example

            .global _start
            .data
        isOn:	.byte  0
            .text
            ...
            global   isOn
            withbool
            uint8   1       ; Test if button on digital input 1 is on
            din
            set             ; save boolean in isOn
            ...
            exit
            .end
