XOR

VM Stack
BeforeAfter
  
bool-2
bool-1bool-1 ⊕ bool-2

Abstract

Remove the two top Boolean values on the stack, and replace them with a single Boolean value representing the logical exclusive or (xor) of the two.

Synopsis

bool-1

bool-2

xor

Arguments

<bool-1> - the first argument (left hand arg from higher level language)

<bool-2> - the second argument (right hand arg from higher level language)

    Returns

    Returns the logical exclusive or (xor) of the two argument expressions.

    Example

            uint8 1         ; Select port 1
        din             ; Get digital input 1
        uint8 2         ; Select port 2
        din             ; Get digital input 2
        xor             ; Are both on?
        block           ; start of if block
        beep            ; Beep if one, and only one, digital input is on
        eob
        if              ; If condition check

    See Also

    and not or
