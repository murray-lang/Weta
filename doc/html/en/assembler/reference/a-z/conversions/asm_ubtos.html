UBTOS

VM Stack
BeforeAfter
  
uint8 valueuint16 value

Abstract

Pop the uint8 (unsigned byte) from the stack and push a uint16 (unsigned short) with the same value.

Synopsis

<uint8>    <value>

ubtos

Where

<value> - the uint8 value to be converted

Note

This synposis shows an immediate value being explicity placed onto the stack to be converted. However, the value could be the result peripheral I/O, function calls or any number of mathematical expressions. Or it could be an arbitrary combination of all of these.

Returns

Returns a uint16 with the value of <value> (by virtue of it being pushed onto the stack).

Example

        ...
            ; Multiply sensor value by 10
        uint8   10
        ubtos
        sensor1     ; sensor values are int16
        withint16   ; treat everything as an int16 (signed)
        mul         ; multiply top two int16s
        tostr       ; convert int16 result to a string
        withstring
        send        ; send to the default stream
        ...
