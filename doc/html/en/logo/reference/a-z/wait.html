WAIT

Wait for the given multiple of 1/10th of a second.

Synopsis

wait <tenths>

Arguments

<tenths> - the number of tenths of a second

    Returns

    Nil.

    Example

                ; Flash the user LED
        ledon
        wait 2  ; wait 2/10ths of a second
        ledoff

    See Also

    onfor waituntil
