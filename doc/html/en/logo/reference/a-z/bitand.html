BITAND

Return the bitwise and of the the two integer arguments.

Synopsis

bitand <lhs> <rhs>

Arguments

<lhs> - Left-hand value. Must be an integer

<rhs> - Right-hand value. Must be an integer

    Returns

    Returns the bitwise and of two arguments. The type of the result is that of the widest argument.

    Example

            make “foo 15                ; TODO: Provide a way to specify values in other radixes (radices?)
        make “bar bitand :foo 7     ; bar == 7
        make “foo bitand :bar 8     ; foo == 0

    bitnot bitor bitxor
