Abstract

Repeat the block of instructions based on the given parameters. Before the block is executed, a counter variable is assigned an initial (int16) from value. Whether the block is executed depends on the result of a comparison between the counter variable and the supplied to value. If step is positive then the block is only executed if the counter is less than or equal to the to value. If step is negative then the block is only executed if the counter is greater than or equal to the to value.

Synopsis

or...

Arguments

<counter> - counter variable name

<from> - from value (int16)

<to> - to value (int16)

<step> - step value (int16)

    Example

                .global _start
            .data
        counter:	.short		true

            .text
        _start:
                ; Countdown
            global   counter    ; address of counter on the stack
            int16    1          ; from
            int16    10         ; to
            int16    1          ; step
            block
            global   counter
            sget                ; get the value of counter
            stostr              ; convert it to a string
            send
            string "\n"
            strsend
            eob
            for
            string "Blastoff!\n"
            strsend
            exit
            .end
