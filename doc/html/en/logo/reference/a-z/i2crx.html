I2CRX

Receive I²C data.

Synopsis

i2crx <i2c-slave-addr> <rx-buffer> <rx-buffer-length> <timeout>

Arguments

<i2c-slave-addr> - the I²C slave address of the source device

<rx-buffer> - reference to a bytearray for rx data

<rx-buffer-length> - the maximum number of bytes to receive

<timeout> - timeout in milliseconds

    Returns

    Nil

    Example

                ...
            bytearray [rxbuff 4]

            i2cstart
            i2crx 1 "rxbuff 4 1000	; Read data
            make "err i2cerr	    ; i2cerr holds the last i2c error
            i2cstop
            ...

    See Also

    i2cerr i2cstart i2cstop i2ctxrx
