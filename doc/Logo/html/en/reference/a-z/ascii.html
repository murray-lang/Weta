ASCII

Return the ASCII value of the given character. If the argument is a string longer than one character then only the first character is considered.

Synopsis

ascii <string-of-length-1>

Arguments

<string-of-length-1> - a character value (eg. 'a')

    Returns

    Returns the ASCII value of the given character.

    Example

            make “foo ascii '\n'     ; foo == 10
