POW

Return the first argument to the power of the second argument.

Synopsis

pow <base> <exponent>

Arguments

<base> - the value to be multiplied by itself

<exponent> - the number of times base is to be multiplied by itself

    Returns

    Returns <base> to the power of <exponent>.

    Example

              make “foo 2
          make “bar pow :foo 3  ; bar == 8
          make “foo pow :bar 2  ; foo == 64

    See Also

    exp sqr sqrt
