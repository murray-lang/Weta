THISWAY

Set the direction of the selected motors. What this direction means depends on the hardware implementation. It will be the opposite direction to thatway.

Synopsis

thisway

Arguments

Nil

    Returns

    Nil.

    Example

                  ; Select motors a and b.
              ;These will stay selected until another selection is made
          ab,
          thisway       ; set the direction of both
          setpower 8    ; full power
          on            ; both on
          wait 10       ; wait 1s
          off           ; both off
          rd            ; reverse direction
          on            ; both on
          wait 10       ; wait 1s
          off           ; both off
          brake         ; make sure both stop turning

    See Also

    brake off on onfor rd setpower thatway
