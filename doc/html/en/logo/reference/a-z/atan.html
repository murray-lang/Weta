ATAN

Find the angle (in radians) that has the given tangent value.

Synopsis

atan <tangent>

Arguments

<tangent> - tangent value to find the angle for

    Returns

    Returns the angle, in radians, that has <tangent> as its tangent.

    Example

            make “foo tan (PI / 4) ; foo == 1
        make “bar atan :foo    ; bar == PI / 4

    See Also
