NEG

Return the value of the argument but with the sign reversed.

Synopsis

neg <number>

Arguments

<number> - the value to be negated

    Returns

    Returns the value of <number> with the sign reversed.

    Example

            make “foo 10
        make "bar neg :foo  ; bar == -10

    See Also

    abs isinf isnan max min
