ERASE

Erase the given number of items from the current location (data point) of the built-in data array.

TODO: Clarify size information and include as a configuration option.

This command is used in conjunction with record, recall and resetdp

Synopsis

erase <num-items>

Arguments

<num-items> - the number of items to erase

    Returns

    Nil.

    Example

                ...
                ; Gather a day's worth of 10 minute data
                ; (To be collected at a later time)
            resetdp
            erase 24 * 6    ; This expression will be resolved at compile time
            repeat 24 * 6
            [
                record sensor1
                wait 10 * 60 * 10
            ]
                ; Wait for the user to press a button connected to switch2
            waituntil [switch2]
            resetdp
            repeat 24 * 6
            [
                send tostring recall
                send '\n'
            ]
            ...

    See Also

    recall record resetdp setdp
