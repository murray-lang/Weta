IFELSE

Execute the first statement block if the given condition evaluates to true. Otherwise execute the second statement block.

Synopsis

ifelse <condition> [ <statements-if-true>* ] [ <statements-if-false>* ]

Where

<condition> - an expression that evaluates to true of false

<statements-if-true> - statements to execute if condition is true

<statements-if-false> - statements to execute if condition is false

    Example

              if digitalin 1
          [
              send 'Digital input 1 is on.\n'
          ]
          [
              send 'Digital input 1 is off.\n'
          ]

    See Also
