SLOT

VM Stack
BeforeAfter
  
slot value

Abstract

Returns the current count within a block of instructions associated with repeat.

The value starts at 1 and increases on each iteration of the block. At the last iteration slot will equal the repeat's count value.

Note

If repeat blocks are nested, then the outer slot value is not available to the inner blocks. If outer iterator values need to be available to inner blocks then consider the for loop instead of repeat.

Synopsis

slot

Arguments

Nil.

Returns

Returns the int16 value of the current iteration count of the innermost repeat block.

Example

        ; countdown
        int16  10    ; Repeat count
        block
        int16  11    ; 11 minus the slot number is the countdown value
        slot         ; Get the current slot (1-10)
        ssub         ; subtract (shorts)
        stostr       ; Convert to a string
        send
        string "\n"
        send
        eob
        repeat
        string "Blastoff!\n"
