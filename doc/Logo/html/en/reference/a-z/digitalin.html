DIGITALIN

Return the value from the given digital input port.

Synopsis

digitalin <port>

Arguments

<port> - evaluates to the logical number of the digital input port

    Returns

    true if the input is "on", otherwise false. (TODO: how to deal with sense?)

    Example

              foreach port [1 3 5 6 9]
          [
              make "portVal 'off'
              if digitalin :port
              [
                  make "portVal 'on'
              ]
              foreach msg ['Port ' :port ' is ' :portVal '\n']
              [
                  send :msg
              ]
          ]

    See Also

    analogin analogout digitalout
