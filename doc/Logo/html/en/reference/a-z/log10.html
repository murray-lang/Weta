LOG10

Return the base 10 logarithm of the argument.

Synopsis

log10 <value>

Arguments

<value> - a positive number

    Returns

    Return the base 10 logarithm of the argument, or NaN if the argument is negative

    Example

              make "foo log10 2        ; foo == 0.3010299956639812
          make "foo log10 1        ; foo == 0
          make "foo log10 10       ; foo == 1
          make "foo log10 10000    ; foo == 4
          make "foo log10 0        ; foo == -infinity
          make "foo log10 -1       ; foo == NaN
