IF

Execute statements in a block if the given condition evaluates to true.

Synopsis

if <boolean-exp> [ <statements>* ]

Where

<boolean-exp> - an expression that evaluates to true of false

<statements> - zero or more program statements

    Example

              if digitalin 1
          [
              send 'Digital input 1 is on.\n'
          ]

    See Also

    do.while for foreach forever ifelse loop repeat while
