OUTPUT

Set the argument as the return value from the current procedure and return execution to the caller.

Any base type can be returned, including strings, but not arrays.

Synopsis

output <value>

Arguments

<value> - value to return

    Returns

    Nil (That is, output cannot be on the right hand side of an assignment)

    Example

            to toDegrees :radians
            output (:radians * PI) / 180
        end

    See Also

    end stop to
