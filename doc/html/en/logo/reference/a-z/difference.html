DIFFERENCE

Return the value that results from subtracting the second argument from the first argument. This is the prefix version of the infix - operator. Neither argument is modified in the process.

Synopsis

difference <minuend> <subtrahend>

or...

<minuend> - <subtrahend>

Arguments

<minuend> - the value being subtracted from

<subtrahend> - the value being subtracted

    Returns

    Return the value that results from subtracting <subtrahend> from <minuend>.

    Example

              make “foo 5
          make “bar difference :foo 2    ; bar == 3
          make “foo :bar - 2             ; foo == 1

    modulo product quotient sum
