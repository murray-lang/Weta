ROUND

Round the given number to the nearest whole number. If the the fractional part of the number is exactly 0.5, then the number is rounded up. The given number is not modified.

Synopsis

round <decimal>

Arguments

<decimal> - the number

    Returns

    Returns the nearest whole number.

    Example

              make “foo 4.4
          make “bar round :foo   ; bar == 4

          make “foo 4.6
          make “bar round :foo   ; bar == 5
