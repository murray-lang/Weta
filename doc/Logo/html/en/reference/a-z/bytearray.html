BYTEARRAY

Create an array of bytes with the given name and size. It was added to the language to support I/O buffers such as required for I2C.

Synopsis

bytearray[ <name> <size>]

Arguments

<name> - name of the array variable

<size> - the maximum number of bytes in the array

    Returns

    Nil.

    Example

              bytearray [rxbuff 4]  ; rxbuff is an array of four bytes
          i2cstart
          make "i2caddr 1
          make "timeout 1000
          i2crx :i2caddr rxbuff 4 :timeout
          i2cstop
          if i2cerr = 0
          [
              for [i 0 3 1]
              [
                  send tostring aget :rxbuff :i
                  send '\n'
              ]
          ]
