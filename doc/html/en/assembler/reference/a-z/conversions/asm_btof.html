BTOF

VM Stack
BeforeAfter
  
int8 valuefloat value

Abstract

Pop the int8 (signed byte) from the stack and push a float with the same value.

Synopsis

<int8>    <value>

btof

Where

<value> - the int8 value to be converted

Note

This synopsis shows an immediate value being explicitly placed onto the stack to be converted. However, the value could be the result peripheral I/O, function calls or any number of mathematical expressions. Or it could be an arbitrary combination of all of these.

Returns

Returns a float with the value of <value> (by virtue of it being pushed onto the stack).

Example

        ...
        int8   -42
        btof
        withfloat   ; treat everything as a float
        tostr       ; convert float value to a string ("-42")
        withstring
        send        ; send "-42" to the default stream
        ...
