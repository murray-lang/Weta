I2CSTOP

Set the I²C driver state to stop. It should be called after an I²C communications session to reset the state machine.

Synopsis

i2cstop

Arguments

Nil

Returns

Nil

Example

            ...
            bytearray [txbuff 4]   ;bytearray is a type specific to this variant of Logo
            bytearray [rxbuff 4]

            ; Put transmit data into the buffer
            aset :txbuff 0 1
            aset :txbuff 1 3
            aset :txbuff 2 5
            aset :txbuff 3 7

            i2cstart
            i2ctxrx 1 "txbuff 4 "rxbuff 4 1000	; Read-thru-write semantics
            make "err i2cerr					; i2cerr holds the last i2c error
            i2cstop
            ...
