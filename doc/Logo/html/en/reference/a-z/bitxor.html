BITXOR

Return the bitwise exclusive or of the the two integer arguments.

Synopsis

bitxor <lhs> <rhs>

Arguments

<lhs> - Left-hand value. Must be an integer

<rhs> - Right-hand value. Must be an integer

    Returns

    Returns the bitwise exclusive or of two arguments. The type of the result is that of the widest argument.

    Example

              make “foo 12               ; TODO: Provide a way to specify values in other radixes (radices?)
          make “bar bitxor :foo 15   ; bar == 3
          make “foo bitxor :bar 3    ; foo == 0

    See Also

    bitand bitnot bitxor
