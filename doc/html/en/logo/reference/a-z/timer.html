TIMER

Returns value of free-running elapsed time device since the last time resett was called. Time units are reported in 1 millisecond counts.

Since the value is a signed short, the maximum time period that can be reported is 32.768 seconds

Synopsis

timer

Arguments

Nil.

Returns

Returns value of free-running elapsed time device since the last time resett was called.

Example

          resett
          waituntil [newserial?]
          make "waitTime timer/1000
          foreach msg ['I waited ' :waitTime ' seconds for that data!\n']
          [
              send :msg
          ]
