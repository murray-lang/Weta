RECALL

Return the value at the current location in the built-in data array, and advance the current location.

TODO: Clarify size information and include as a configuration option.

This command is used in conjunction with erase, record and resetdp

Synopsis

recall

Arguments

Nil

Returns

The (signed short) value contained at the current location in the built-in data array

Example

            ...
                ; Gather a day's worth of 10 minute data
                ; (To be collected at a later time)
            resetdp
            erase 24 * 6    ; This expression will be resolved at compile time
            repeat 24 * 6
            [
                record sensor1
                wait 10 * 60 * 10
            ]
                ; Wait for the user to press a button connected to switch2
            waituntil [switch2]
            resetdp
            repeat 24 * 6
            [
                send tostring recall
                send '\n'
            ]
            ...

See Also

erase record resetdp setdp
