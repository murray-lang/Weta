REPEAT

VM Stack

Abstract

Repeat the block of instructions the given number of times.

The 1-based current count is available within the block through the slot instruction.

Synopsis

int16    <count>

block

...

eob

repeat

Arguments

<count> - number of times to repeat (int16)

    Example

                ; countdown
        int16  10    ; Repeat count
        block
        int16  11    ; 11 minus the slot number is the countdown value
        slot         ; Get the current slot (1-10)
        ssub         ; subtract (shorts)
        stostr       ; Convert to a string
        send
        string "\n"
        strsend
        eob
        repeat
        string "Blastoff!\n"
        strsend
