NOT

Return the logical complement of the argument.

Synopsis

not <bool-exp>

Arguments

A single argument that must evaluate to a Boolean.

Returns

Returns the logical complement of the argument expression.

Example

        make “foo not true          ; foo == false
        make “foo not false         ; foo == true
        make “foo false
        make “bar not and :foo true ; bar == true

See Also

and or xor
