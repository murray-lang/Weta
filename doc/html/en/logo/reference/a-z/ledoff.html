LEDOFF

Turn off the user LED.

Synopsis

ledoff

Arguments

Nil

    Returns

    Nil.

    Example

                  ; Flash the LED in an approximately one second cycle
          forever
          [
              ledon
              wait 1
              ledoff
              wait 9
          ]
