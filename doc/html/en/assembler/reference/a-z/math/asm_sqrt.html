MATH.SQRT

VM Stack
BeforeAfter
  
value√value

Abstract

Remove the top numerical value from the stack, and push the square root of the value.

The data type expected by math.sqrt is determined by the last with<type> instruction. The valid options for math.sqrt are: withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling math.sqrt is to use one of the prefixed versions of the the instruction, namely:

math.fsqrt or math.dsqrt

The assembler will expand these to a corresponding with<type> instruction followed by math.sqrt.

Synopsis

value

math.sqrt

Arguments

<value> - the value to find the square root of

    Note

    This synopsis shows an immediate value being explicitly placed onto the stack. However, the value could be the result peripheral I/O, function calls or any number of mathematical expressions. Or it could be an arbitrary combination of all of these.

    Returns

    Returns the square root of the argument expression..
