ANALOGOUT

Output the given value to the given analog output port.

Synopsis

analogout <port> <value>

Arguments

<port> - evaluates to the logical number of the analog output port

<value> - evaluates to the value to output

    Returns

    Nil.

    Example

                  ; Output a sinewave. Assume unsigned 8 bit analog output
          make "port 1
          for [i 0 255 1]
          [
              make "outval sin (2 * PI * :i) / 255
              analogout :port :outval               ; outval will be cast to a byte
          ]
