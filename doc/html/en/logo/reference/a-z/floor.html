FLOOR

Round the given number down to the next lowest whole number. The given number is not modified.

Synopsis

floor <decimal>

Arguments

<decimal> - the number

    Returns

    Returns the next lowest whole number.

    Example

            make “foo 4.9
        make “bar floor :foo   ; bar == 4 (even though the nearest whole number is 5)

    See Also

    ceil round
