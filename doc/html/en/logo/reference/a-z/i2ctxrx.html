I2CTXRX

Transmit a buffer through I²C and receive a response buffer in a single operation.

Synopsis

i2ctxrx <i2c-slave-addr> <tx-buffer> <tx-buffer-length> <rx-buffer> <rx-buffer-length> <timeout>

Arguments

<i2c-slave-addr> - the I²C slave address of the source device

<tx-buffer> - reference to a bytearray with tx data

<tx-buffer-length> - number of bytes to send

<rx-buffer> - reference to a bytearray for rx data

<rx-buffer-length> - the maximum number of bytes to receive

<timeout> - timeout in milliseconds

    Returns

    Nil

    Example

                ...
            bytearray [txbuff 4]   ;bytearray is a type specific to this variant of Logo
            bytearray [rxbuff 4]

            ; Put transmit data into the buffer
            aset :txbuff 0 1
            aset :txbuff 1 3
            aset :txbuff 2 5
            aset :txbuff 3 7

            i2cstart
            i2ctxrx 1 "txbuff 4 "rxbuff 4 1000	; Read-thru-write semantics
            make "err i2cerr					; i2cerr holds the last i2c error
            i2cstop
            ...

    See Also

    i2cerr i2crx i2cstart i2cstop
