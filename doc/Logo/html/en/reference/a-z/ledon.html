LEDON

Turn on the user LED.

Synopsis

ledon

Arguments

Nil

    Returns

    Nil.

    Example

                  ; Flash the LED in an approximately one second cycle
          forever
          [
              ledon
              wait 1
              ledoff
              wait 9
          ]
