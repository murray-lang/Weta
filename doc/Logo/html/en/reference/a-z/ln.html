LN

Return the natural logarithm (base e) of the argument.

Synopsis

ln <value>

Arguments

<value> - a positive number

    Returns

    Return the natural logarithm (base e) of the argument, or NaN if the argument is negative

    Example

              make "foo ln E        ; foo == 1
          make "foo ln 1        ; foo == 0
          make "foo ln 10       ; foo == 2.302585092994046
          make "foo ln 0        ; foo == -infinity
          make "foo ln -1       ; foo == NaN

    See Also

    exp ln log10
