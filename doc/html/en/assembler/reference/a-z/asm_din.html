DIN

VM Stack
BeforeAfter
  
uint8 port#bool value

Abstract

Remove the uint8 port number from the top of the stack, and replace it with the Boolean value read from that digital input port.

Synopsis

uint8    port#

din

Arguments

<port#> - the required port number (uint8)

Returns

Returns the Boolean value from the specified digital port.

Example

            ; Wait for a button press. Beep when it happens
            block
            uint8 1    ; assume a button is connected to digital input 1
            din
            eob
            waituntil
            beep
