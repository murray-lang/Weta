REPEAT

Execute statements in a block the given number of times. repcount can be used to access the current number. repcount is 1-based.

Synopsis

repeat <times> [ <statements>* ]

Where

<times> - the number of times to execute the block statements

<statements> - zero or more program statements

    Example

                  ; send '1' to '5'on their own lines
          repeat 5
          [
              send tostring repcount
              send '\n'
          ]

    See Also

    do.while for foreach forever if ifelse loop while
