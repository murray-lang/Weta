MATH.COSH

VM Stack
BeforeAfter
  
anglehyperbolic cosine of angle

Abstract

Remove the angle (in radians) value from the stack, and push the hyperbolic cosine of the angle.

The data type expected by math.cosh is determined by the last with<type> instruction. The valid options for math.cosh are: withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling cosh is to use one of the prefixed versions of the the instruction, namely:

math.fcosh or math.dcosh

The assembler will expand these to a corresponding with<type> instruction followed by math.cosh.

Synopsis

angle

math.cosh

Arguments

<angle> - the angle (in radians) to find the hyperbolic cosine of

    Note

    This synopsis shows an immediate value being explicitly placed onto the stack. However, the value could be the result peripheral I/O, function calls or any number of mathematical expressions. Or it could be an arbitrary combination of all of these.

    Returns

    Returns the hyperbolic cosine of the given angle.
