AOUT

BeforeAfter
  
uint8 port#
uint8 value

Table 1. VM Stack


Remove the uint8 port number and uint8 value from the top of the stack, and send the value to the analog output port identified by port#.

Synopsis

value

port#

aout

Arguments

<value> - value to output (uint8)

<port#> - the required port number (uint8)

    Returns

    Nothing is returned to the stack.

    Example

            uint8 1         ; #1
        ain             ; Get user knob setting from analog input 1
        itoub           ; convert the int16 to a uint8 value to output
        uint8 2         ; #2
        aout            ; Send the knob setting to the LED at analog output 2 (brightness)

    See Also
