ATAN2

Find the angle (in radians) between the positive x-axis and the point given by coordinates x and y. Both x and y can be positive or negative, and therefore identify a point in any quadrant.

Synopsis

atan <x> <y>

Arguments

<x> - x coordinate of the point

<y> - y coordinate of the point

    Returns

    Returns the angle, in radians, between the x-axis and (<x>, <y>).

    Example

              make “foo atan 1 1    ; foo == PI / 4
          make “foo atan 0 1    ; foo == PI / 2
          make “foo atan -1 0   ; foo == PI
          make “foo atan 0 -1   ; foo == 3 * PI / 2
