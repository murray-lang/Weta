STOP

Returns execution from a procedure to the caller.

stop is not required at the end of a procedure definition as it is implied by end. It is provided as a short-cut for exiting a procedure as the result of some condition.

Synopsis

stop

Example

                ; Wait for the given digital input to be the given state
            to waitForDI :port :state
                forever
                [
                    if digitalin :port = :state
                        return
                    wait 1
                ]
            end
