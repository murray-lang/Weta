WITHINT8

Abstract

Selects int8 as the type to be used for any subsequent stack operations.

Synopsis

withint8

Arguments

Nil.

Returns

Nil. (The instruction sets a register that is internal to the virtual machine.)

Example

            .global _start
            .data
        val:	.byte  0
            .text
            ...
            global   val    ; increment val
            withint8
            get
            int8   -1       ; int8 is signed
            sub             ; increment by subtracting -1
            global   val
            set
            ...
            exit
            .end
