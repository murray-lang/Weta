SWITCH[1-8]

Returns the Boolean value of the switch (digital input) identified by the suffix digit. These commands are included for compatibility with Cricket Logo, but the same functionality is provided in a more generic way by digitalin.

On the Gogo board (which this Cricket Logo is derived from) switches 1-8 are in fact sensors 1-8, but being connected to a switch means that the analogue values are at either the maximum or minimum. So the switch 1-8 commands read the analog value and convert it to digital based on a threshold test. Assume that this is the case unless you have documentation for your target device or firmware that states otherwise.

Synopsis

switch1
switch2

...

switch8

Arguments

Nil.

Returns

Returns the Boolean value of the switch identified by the suffix digit.

Example

            ; Send some switch data
        foreach ip [switch1 ', ' switch2 ', ' switch3 ', ' switch4 '\n']
        [
                ; Because at least one list item is a string, all values are
                ; converted to a string. Since the switch values are Boolean,
                ; they will be converted to 'true' or 'false'.
            send :ip
        [
