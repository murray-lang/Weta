INT8

VM Stack
BeforeAfter
  
uint8 value

Abstract

Push the given number onto the stack as a signed 8 bit value.

Synopsis

int8    <value>

Arguments

<value> - a number between -127 and 127 (int8)

Returns

Returns a signed 8 bit number (by virtue of it being pushed onto the stack)

Example

        int8   5
        int8   -6
        btostr          ; Replace -30 on the stack with "-30"
        strsend         ; Send the string to the default output stream
