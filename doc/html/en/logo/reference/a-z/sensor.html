SENSOR[1-8]

Returns value of the analog sensor identified by the suffix digit. These commands are included for compatibility with Cricket Logo, but the same functionality is provided in a more generic way by analogin.

The value is a signed short, but the range of values will depend on the ADC hardware. In Cricket Logo the range is specified as from 0 to 1023 (Assumes 10 bit ADC)

Synopsis

sensor1
sensor2

...

sensor8

Arguments

Nil.

Returns

Returns value of the analog sensor identified by the suffix digit.

        foreach ip [sensor1 ', ' sensor2 ', ' sensor3 ', ' sensor4 '\n']
