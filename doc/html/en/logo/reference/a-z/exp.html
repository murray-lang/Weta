EXP

Return e (Euler's constant) to the power of the given exponent.

Synopsis

exp <exponent>

Arguments

<exponent> - the power to which e should be raised

    Returns

    Return e to the power of the given exponent.

    Example

              make "foo E           ; foo == 2.718281828459045
          make "foo exp 1       ; foo == 2.718281828459045
          make "foo exp -1      ; foo == 0.36787944117144233
          make "foo exp 0       ; foo == 1

    See Also

    E ln log10 pow sqr sqrt
