E

Return the constant e (Euler's constant).

Synopsis

E

Arguments

Nil

Returns

Returns the constant e (Euler's constant).

Example

          make "foo E           ; foo == 2.718281828459045
          make "foo exp 1       ; foo == 2.718281828459045
          make "foo exp -1      ; foo == 0.36787944117144233
          make "foo exp 0       ; foo == 1
