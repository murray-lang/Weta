ABS

Return the absolute value of the argument.

Synopsis

abs <numeric-exp>

Arguments

<numeric-exp> - must evaluate to a number

    Returns

    Returns the absolute value of the argument expression.

    Example

            make “foo -42       ; foo == -42
        make “bar abs :foo  ; bar == 42

    See Also

    isinf isnan min neg
