DIGITALOUT

Set or reset the given digital output port.

Synopsis

digitalout <port> <value>

Arguments

<port> - evaluates to the logical number of the analog output port

<value> - boolean value of the port

    Returns

    Nil.

    Example

                  ; give an indication if an analog input is above a threshold
          make "indicatorPort 1
          make "analogPort 8
          make "threshold  127
          forever
          [
              digitalout :indicatorPort (analogin :analogPort > :threshold)
              wait 1
          ]

    See Also

    analogin analogout digitalin
