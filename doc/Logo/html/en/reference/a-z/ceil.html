CEIL

Round the given number up to the next higher whole number (ie. ceiling). The given number is not modified.

Synopsis

ceil <decimal>

Arguments

<decimal> - the number

    Returns

    Returns the next highest whole number.

    Example

            make “foo PI
        make “bar ceil :foo   ; bar == 4 (even though the nearest whole number is 3)

    See Also

    floor round
