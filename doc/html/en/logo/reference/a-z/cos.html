COS

Return the cosine of the given angle (expressed in radians).

Synopsis

cos <radians>

Arguments

<radians> - the angle in radians

    Returns

    Returns the cosine of the given angle.

    Example

            make “foo cos 0       ; foo == 1
        make “bar acos :foo   ; bar == 0

    See Also

    acos asin atan atan2 cosh sin sinh tan tanh
