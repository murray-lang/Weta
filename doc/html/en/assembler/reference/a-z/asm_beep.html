BEEP

Abstract

Cause a short beep from the standard piezo buzzer (or equivalent).

Synopsis

beep

Arguments

Nil.

Returns

Nil.

Example

            ; Wait for a button press. Beep when it happens
        block
        uint8 1    ; assume a button is connected to digital input 1
        din
        eob
        waituntil
        beep

See Also

ledoff ledon waituntil
