ISNAN

Return true if the argument is Not a Number (NaN), otherwise false.

Synopsis

isnan <floating-point-num>

Arguments

<floating-point-num> - the number to test

    Returns

    Return true if <floating-point-num> is Not a Number (NaN).

    Example

            make "foo ln -1        ; foo == NaN
        ifelse isnan :foo
        [
            send 'foo is NaN!\n'
        ]
        [
            send 'BUG! something is not right with ln or isinf\n'
        ]

    See Also

    abs isinf max min neg
