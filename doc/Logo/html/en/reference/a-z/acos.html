ACOS

Find the angle (in radians) that has the given cosine value.

Synopsis

acos <cosine>

Arguments

<cosine> - cosine value to find the angle for

    Returns

    Returns the angle, in radians, that has <cosine> as its cosine.

    Example

            make “foo cos 0       ; foo == 1
        make “bar acos :foo   ; bar == 0

    See Also

    asin atan atan2 cos cosh sin tanh tan tanh
