IFELSE

Abstract

Execute one block of code if the given condition test evaluates to true, and a different block if it evaluates to false.

Synopsis

block

<if-instructions>

eob

block

<else-instructions>

eob

<condition-test>

ifelse

Arguments

<if-instructions> - instructions to execute if the condition is true

<else-instructions> - instructions to execute if the condition is false

<condition-test> - instructions that evaluate to a boolean (bool)

    Example

                ; LED follows digital input 3
        block         ; forever block
        block         ; if block
        ledon
        eob
        block         ; else block
        ledoff
        eob
        uint8   3     ; get digital input 3
        din
        ifelse        ; test digital input result (bool)
        eob
        forever
