ASHIFT

Returns the given value shifted by the given number of bits (the argument is not modified). If the number of bits is positive then the value will be shifted left. If the number of bits is negative then the value will be shifted right. The sign bit will be extended with a right shift. (The 'A' in ASHIFT stands for 'arithmetic', and indicates this retention of sign information)

Synopsis

ashift <value> <number-of-bits>

Arguments

<value> - the value to shift. Must be an integer value

<number-of-bits> - number of bits to shift (-ve for right shift)

    Returns

    Returns <value> shifted by the given number of bits.

    Example

              make "foo 1
          make "bar ashift :foo 1    ; bar == 2
          make "bar ashift :foo 2    ; bar == 4
          make "foo ashift :bar -2   ; foo == 1

          make "foo -1
          make "bar ashift :foo -1   ; bar == -1 (still)
          make "bar ashift :foo -2   ; bar == -1 (still)
          make "bar ashift :foo 1    ; bar == -2
          make "bar ashift :foo 2    ; bar == -4
