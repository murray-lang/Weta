STRING

Create a string with the given name and size. The syntax is borrowed from the FMS Logo syntax for array declarations.

Strings can be created by simply assigning a literal string with make. However, using the method described on this page is more efficient where a string variable will later be set by a run-time function. It is the responsibility of the programmer to set a string to a sufficient size for the purpose to which it will be used.

Strings are most likely stored internally as 0 terminated. There is no need to add this extra character to the string size as it will be done automatically by the compiler.

Synopsis

make "<name> (string <size>)

Arguments

<name> - name of the string variable

<size> - the maximum number of items in the array

    Returns

    Nil.

    Example

                  ; The first assignment creates the variable.
              ; In this case the size is being set to 5 so that it can hold the
              ; string representation of any short value.
              ; (shorts are the default integer size)
          make "str (string 5)
          make "str tostring 32767
          send :str

    See Also

    array make tostring
