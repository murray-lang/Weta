OR

Return the logical or of the the two boolean arguments.

Synopsis

or <bool-exp> <bool-exp>

Arguments

Exactly two arguments are required, and they must evaluate to Booleans.

Returns

Returns the logical or of the two argument expressions.

Example

        make “foo or false false   ; foo == false
        make “foo or true false    ; foo == true
        make “foo false
        make “bar or :foo true     ; bar = true

See Also

and xor not
