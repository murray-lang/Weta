Table 1. VM Stack


begin must be the first instruction in a procedure. It saves the machine context of the caller, before setting up a context frame for the procedure being defined.

While it uses the VM stack to store this information, it will be specific to the firmware implementation and is not relevant for the purpose of the general processing instructions (other than being prone to corruption by unbalanced stack operations).

Synopsis

begin

Arguments

None

Returns

Nothing is returned to the stack for general use.

Example

        myBeep  .proc
            begin
            beep
            return
            .endproc
