OR

VM Stack
BeforeAfter
  
bool-2
bool-1bool-1 ⋁ bool-2

Abstract

Remove the two top Boolean values on the stack, and replace them with a single Boolean value representing the logical or of the two.

Synopsis

bool-1

bool-2

or

Arguments

<bool-1> - the first argument (left hand arg from higher level language)

<bool-2> - the second argument (right hand arg from higher level language)

    Returns

    Returns the logical or of the two argument expressions.

    Example

            uint8 1         ; Select port 1
        din             ; Get digital input 1
        uint8 2         ; Select port 2
        din             ; Get digital input 2
        or             ; Are both on?
        block           ; start of if block
        beep            ; Beep if either digital inputs are on
        eob
        if              ; If condition check

    See Also

    and not xor
