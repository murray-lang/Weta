MAX

Return the highest value contained in either of the two arguments. The type of the returned value is the same as the widest of the arguments.

Synopsis

max <number1> <number2>

Arguments

<number1> - one of the numbers

<number2> - the other number

    Returns

    Returns the highest value contained in either <number1> or <number2>.

    Example

              make “foo 5
          make “bar max :foo 8    ; bar == 8

    See Also

    abs isinf isnan min neg
