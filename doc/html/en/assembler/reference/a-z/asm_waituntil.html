WAITUNTIL

Abstract

Wait until the code in a block returns a true.

Synopsis

block

<instructions>

eob

waituntil

Arguments

<instructions> - code that ultimately pushes a Boolean value onto the stack

    Example

                ; Wait for a button press. Beep when it happens
            block
            uint8 1    ; assume a button is connected to digital input 1
            din
            eob
            waituntil
            beep
