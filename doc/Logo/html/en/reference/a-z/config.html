CONFIG

The config is used to configure a (growing) range of functions in Weta Logo. In general it can be called at any time in a program, though in most cases it will be called once at the beginning of a program to specify some deviation from from default behaviour.

Synopsis

config digitalin [<port-list>...;]
config digitalout [<port-list>...;]
config send serial <port>
config send ethernet (Not yet implemented)
config serial <port> [ baud:<baud-rate> databits:5|6|7|8 stopbits:1|2 parity:odd|even|none ]

Arguments

digitalin

<port-list> - a list of logical port numbers which are to be digital inputs

    digitalout

    <port-list> - a list of logical port numbers which are to be digital outputs

      send serial

      <port> - the default logical serial port number to use for the send command

        send ethernet

        Note: Ethernet output for send has not yet been implemented.

        serial

        <port> - the logical serial port for which the following parameters are to apply

        <baud-rate> - the required baud rate

          Returns

          Nil.

          Example

                      ; Provide a list of logical digital I/O ports that will be inputs
        config digitalin [1 2 3 4]
            ; Provide a list of logical digital I/O ports that will be outputs
        config digitalout [5 6 7 8]
            ; Specify the stream type and port to be used by the default send command.
            ; Alternatives at the time of writing are 'serial' and 'ethernet'.
            ; The port numbers are 1-based
        config send serial 1
            ; Configure the parameters of the given serial port
        config serial 1 [baud:9600 databits:8 stopbits:1 parity:none]
