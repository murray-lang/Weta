PRODUCT

Return the value that results from multiplying the two arguments. This is the prefix version of the infix * operator. Neither argument is modified in the process.

Synopsis

sum <multiplicand> <multiplier>

or...

<multiplicand> * <multiplier>

Arguments

<multiplicand>

<multiplier>

    Returns

    Return the value that results from multiplying <multiplicand> to <multiplier>.

    Example

                  make “foo 5
              make “bar product :foo 2    ; bar == 10
              make “foo :bar * 2          ; foo == 20

    See Also

    difference modulo quotient sum
