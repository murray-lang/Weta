NE

VM Stack
BeforeAfter
  
value-2
value-1value-1 ≠ value-2 (bool)

Abstract

Remove the two top values from the stack, and push false if the values are equal, otherwise push true.

The data types expected by ne are determined by the last with<type> instruction. The valid options for ne are: withbool, withint8, withuint8, withint16, withuint16, withint32, withuint32, withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling ne is to use one of the prefixed versions of the the instruction, namely:

qne (bool), bne, ubne, sne, usne, ine, uine, fne and dne

The assembler will expand these to a corresponding with<type> instruction followed by ne.

Synopsis

value-1

value-2

ne

Arguments

<value-1> - the first argument (left hand arg from higher level language)

<value-2> - the second argument (right hand arg from higher level language)

    Returns

    Returns true if the two values are not equal, false if they are equal.
