LSHIFT

Returns the given value shifted by the given number of bits (the argument is not modified). If the number of bits is positive then the value will be shifted left. If the number of bits is negative then the value will be shifted right. The sign bit will NOT be extended with a right shift. (The 'L' in LSHIFT stands for 'logical', and indicates that sign is not considered)

Synopsis

lshift <value> <number-of-bits>

Arguments

<value> - the value to shift. Must be an integer value

<number-of-bits> - number of bits to shift (-ve for right shift)

    Returns

    Returns <value> shifted by the given number of bits.

    Example

              make "foo 1
          make "bar lshift :foo 1    ; bar == 2
          make "bar lshift :foo 2    ; bar == 4
          make "foo lshift :bar -2   ; foo == 1

          make "foo -1
          make "bar lshift :foo -1   ; bar == 32767
          make "bar lshift :foo -2   ; bar == 16383
          make "bar lshift :foo 1    ; bar == -2
          make "bar lshift :foo 2    ; bar == -4
