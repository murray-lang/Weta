VM Stack
BeforeAfter
  
num-items (uint16)

Abstract

Pop the uint16 value at the top of the stack and erase that number of items starting at the current location in the internal data array.

The data pointer can be reset to 0 using resetdp, and set to an explicit location using setdp.

Synopsis

<num-items>

erase

Arguments

<num-items> - the number of items to erase (uint16)

    Returns

    Nil.

    Example

            ...
            ; Record 1 minute's data from sensor1 at 1 second intervals
        resetdp       ; reset the data pointer to 0
        uint16    60
        erase
        uint16   60   ; repeat count
        block
        sensor1       ; sensor1 value (int16) on the stack
        record        ; store it
        uint16   1000 ; wait 1 second
        wait
        eob
        repeat
        ...

    See Also

    recall record resetdp setdp
