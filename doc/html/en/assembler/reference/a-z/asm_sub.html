Abstract

Remove the two top numerical values from the stack, and push the value calculated by subtracting the top value from the second top value.

The data type expected by add is determined by the last with<type> instruction. The valid options for sub are: withint8, withuint8, withint16, withuint16, withint32, withuint32, withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling add is to use one of the prefixed versions of the the instruction, namely:

bsub, ubsub, ssub, ussub, isub, uisub, fsub and dsub

The assembler will expand these to a corresponding with<type> instruction followed by sub.

Synopsis

value-1

value-2

add

Arguments

<value-1> - the first argument (left hand arg from higher level language)

<value-2> - the second argument (right hand arg from higher level language)

    Returns

    Returns <value-1> minus <value-2>.
