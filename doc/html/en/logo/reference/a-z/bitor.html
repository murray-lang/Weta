BITOR

Return the bitwise or of the the two integer arguments.

Synopsis

bitor <lhs> <rhs>

Arguments

<lhs> - Left-hand value. Must be an integer

<rhs> - Right-hand value. Must be an integer

    Returns

    Returns the bitwise or of two arguments. The type of the result is that of the widest argument.

    Example

              make “foo 0                 ; TODO: Provide a way to specify values in other radixes (radices?)
          make “bar bitor :foo 8     ; bar == 8
          make “foo bitor :bar 4     ; foo == 12
          make “bar bitor :foo 9     ; bar == 13
