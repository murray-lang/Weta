ISINF

Return true if the argument is infinity (positive or negative), otherwise false.

Synopsis

isinf <floating-point-num>

Arguments

<floating-point-num> - the number to test

    Returns

    Return true if <floating-point-num> is positive or negative infinity.

    Example

            make "foo ln 0        ; foo == -infinity
        ifelse isinf :foo
        [
            send 'foo is infinity!\n'
        ]
        [
            send 'BUG! something is not right with ln or isinf\n'
        ]

    See Also

    abs isnan max min neg
