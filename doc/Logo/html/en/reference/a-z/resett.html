RESETT

Resets the free-running timer to 0.

This command is used in conjunction with timer

Synopsis

resett

Arguments

Nil.

Returns

Nil.

Example

          resett
          waituntil [newserial?]
          make "waitTime timer/1000
          foreach msg ['I waited ' :waitTime ' seconds for that data!\n']
          [
              send :msg
          ]
