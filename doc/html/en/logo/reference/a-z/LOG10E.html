LOG10E

Return the base 10 logarithm of e (Euler's constant).

Synopsis

LOG10E

Arguments

Nil

Returns

Returns the base 10 logarithm of e (Euler's constant)

Example

          make "foo LOG10E    ; foo == 0.4342944819032518
