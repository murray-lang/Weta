BITNOT

Return the bitwise complement of the argument.

Synopsis

bitnot <rhs>

Arguments

<rhs> - Value to invert. Must be an integer

    Returns

    Returns the bitwise complement of the arguments. (The bits are inverted)

    Example

              make “foo -1               ; TODO: Provide a way to specify values in other radixes (radices?)
          make “bar bitnot :foo    ; bar == 0
          make “foo 65534
          make "bar bitnot :foo    ; bar == 1   (assuming unsigned 16 bit)

    See Also

    bitand bitor bitxor
