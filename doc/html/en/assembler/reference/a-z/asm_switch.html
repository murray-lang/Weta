SWITCH[1-8]

VM Stack
BeforeAfter
  
Boolean value

Abstract

Push the value obtained from a switch onto the stack. The switch number is determined by the suffix digit of the instruction. (i.e. switch1, switch2 ... switch8)

Note

The switch[1-8] instructions actually interpret the analog inputs obtained through the corresponding sensor[1-8] instructions. Boolean on/off values are gleaned from the extreme values resulting from placing switches (with pullups) on the sensor inputs. This behaviour is inherited from the Gogo board, and is a solution to the problem of providing a choice of analog or digital inputs when available I/O ports are limited.

Synopsis

switch1

switch2

...

switch8

Arguments

Nil.

Returns

Returns a boolean value corresponding to the switch state.
