MODULO

Return the remainder from the division of the first argument by the second argument. This is the prefix version of the infix % operator. Neither argument is modified in the process.

Synopsis

modulo <dividend> <divisor>

or...

<dividend> % <divisor>

Arguments

<dividend> - the value to be divided

<divisor> - the number of divisions

    Returns

    Returns the remainder from the division of <dividend> by <divisor>.

    Example

              make “foo 14
          make “bar modulo :foo 5    ; bar == 4
          make “foo :bar % 3         ; foo == 1
