output removes the value from the top of the stack and puts it into the stack location just before the procedure's arguments. It is assumed that the caller will clean up these arguments (if any), leaving the returned value on top of the stack for a subsequent instruction to consume.

Synopsis

return

Arguments

None

Returns

Nothing is returned to the stack for general use.

Example

        myBeep  .proc
            begin
            beep
            return
            .endproc
