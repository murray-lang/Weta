TANH

Return the hyperbolic tangent of the given angle (expressed in radians).

Synopsis

tanh <radians>

Arguments

<radians> - the angle in radians

    Returns

    Returns the hyperbolic tangent of the given angle.

    Example

                ; Verify trig identity
          make “x 0
          make "result1 sinh :x / cosh :x
          make "result2 tanh :x
          ifelse result1 = result2
          [
              send 'tanh x = sinh x / cosh x as expected.\n'
          ]
          [
              send 'Something is not right with sinh, cosh and/or tanh.\n'
          ]

    See Also
