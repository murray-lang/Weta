WHILE

Execute statements in the block until the condition becomes true. If the condition is not true when the statement is reached then the block statements will never be executed.

Synopsis

while <condition> [ <statements>* ]

Where

<condition> - must evaluate to a Boolean value

<statements> - zero or more program statements

    Example

                  ; wait for a digital input before continuing
          while not digitalin 3
          [
              wait 1
          ]
          beep

    See Also

    while for foreach forever if ifelse loop repeat
