LE

VM Stack
BeforeAfter
  
value-2
value-1value-1 ≤ value-2 (bool)

Abstract

Remove the two top values from the stack, and push true if the second top value is less than or equal to the top value, otherwise push false.

The data types expected by le are determined by the last with<type> instruction. The valid options for le are: withint8, withuint8, withint16, withuint16, withint32, withuint32, withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling le is to use one of the prefixed versions of the the instruction, namely:

ble, uble, sle, usle, ile, uile, fle and dle

The assembler will expand these to a corresponding with<type> instruction followed by le.

Synopsis

value-1

value-2

le

Arguments

<value-1> - the first argument (left hand arg from higher level language)

<value-2> - the second argument (right hand arg from higher level language)

    Returns

    Returns true if value-1 is less than or equal to value-2, otherwise false.
