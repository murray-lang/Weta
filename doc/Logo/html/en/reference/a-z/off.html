OFF

Turn off the selected motors.

Synopsis

off

Arguments

Nil

    Returns

    Nil.

    Example

                  ; Select motors a and b.
              ;These will stay selected until another selection is made
          ab,
          thisway       ; set the direction of both
          setpower 7    ; full power
          on            ; both on
          wait 10       ; wait 1s
          off           ; both off
          rd            ; reverse direction
          on            ; both on
          wait 10       ; wait 1s
          off           ; both off
          brake         ; make sure both stop turning

    See Also

    brake on onfor rd setpower thatway thisway
