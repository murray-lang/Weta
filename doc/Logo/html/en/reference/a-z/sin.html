SIN

Return the sine of the given angle (expressed in radians).

Synopsis

sin <radians>

Arguments

<radians> - the angle in radians

    Returns

    Returns the sine of the given angle.

    Example

            make “foo sin (PI / 2.0)     ; foo == 1
        make “bar asin :foo          ; bar == PI / 2

    See Also
