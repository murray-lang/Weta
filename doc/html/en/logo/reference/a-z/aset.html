ASET

Set an element in an array.

Synopsis

aset <array-var> <index-exp> <value>

Arguments

<array-var> - array to retrieve the element from

<index-exp> - index of the array element

<value> - the value to assign to the array element

    Returns

    Nil.

    Example

              array [foo 4]

          aset :foo 0 42				; initialise the array
          aset :foo 1 (aget :foo 0) + 1
          aset :foo 2 (aget :foo 1) + 1
          aset :foo 3 (aget :foo 2) + 1
              ; send “42 43 44 45 ” to the default stream
          for [i 0 3 1]
          [
              send tostring aget :foo :i
              send ' '
          ]

    See Also

    array setitem aget item
