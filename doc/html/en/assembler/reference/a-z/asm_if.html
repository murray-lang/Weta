IF

Abstract

Execute a block of code if the given condition test evaluates to true.

Synopsis

block

...

eob

<condition-test>

if

Arguments

<condition-test> - instructions that evaluate to a boolean (bool)

    Example

                ; beep if switch 1 is on
        block
        beep
        eob
        switch1
        if
