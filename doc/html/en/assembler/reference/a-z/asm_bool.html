BeforeAfter
  
bool value

Abstract

Push the given boolean value onto the stack. In the current VM firmware, Booleans are represented by bytes.

Synopsis

bool    true | false

Arguments

Either true or false

Returns

Returns a bool (by virtue of it being pushed onto the stack)
