DO.WHILE

Execute statements in the block until the condition becomes true. The block statements will always be executed at least once before the condition is tested.

Synopsis

do.while <condition> [ <statements>* ]

Where

<condition> - must evaluate to a Boolean value

<statements> - zero or more program statements

    Example

                  ; wait for an analog input to fall below a threshold
          make "threshold 20
          make "val analogin 3
          do.while not :val > :threshold
          [
              make "val analogin 3
          ]
          beep

    See Also

    for foreach forever if ifelse loop repeat while
