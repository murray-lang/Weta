END

Marks the end of a procedure definition (started with to)

Synopsis

to <procedure-name> <statements>* end

Where

<procedure-name> - the name of the procedure being defined

<statements> - the procedure's statements

    Example

                    ; Define a procedure that flashes the user LED
            to flash
                ledon
                wait 1
                ledoff
            end

    See Also

    output stop to
