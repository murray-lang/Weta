BLOCK

VM Stack
BeforeAfter
  
code address of first instruction

Abstract

Marks the beginning of a block of code as used by the flow control structures. (The end is marked by eob.)

The byte code for block is followed by a number - length of the block - and this is used by the firmware to calculate the jump distance. However, the assembler doesn't require this number as it is calculated using the location of the next occurring eob.

The code address pushed onto the stack (as indicated by the table) is consumed by the flow control instruction that immediately follows the eob.

Synopsis

block

Arguments

Nil.

Returns

Nothing is returned to the stack for general use.
