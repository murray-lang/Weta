UINT8

VM Stack
BeforeAfter
  
uint8 value

Abstract

Push the given number onto the stack as an unsigned 8 bit value.

Synopsis

uint8    <value>

Arguments

<value> - a number between 0 and 255 (uint8)

Returns

Returns an unsigned 8 bit number (by virtue of it being pushed onto the stack)

Example

        uint8 3         ; Select port 3
        ain             ; Get analog input 3
        itostr          ; Replace the int16 with its string representation
        strsend         ; Send the string to the default output stream
