Table 1. VM Stack


Remove the two top Boolean values on the stack, and replace them with a single Boolean value representing the logical and of the two.

Synopsis

bool-1

bool-2

and

Arguments

<bool-1> - the first argument (left hand arg from ligher level language)

<bool-2> - the second argument (right hand arg from ligher level language)

    Returns

    Returns the logical and of the two argument expressions.

    Example

            uint8 1         ; Select port 1
        din             ; Get digital input 1
        uint8 2         ; Select port 2
        din             ; Get digital input 2
        and             ; Are both on?
        block           ; start of if block
        beep            ; Beep if both digital inputs are on
        eob
        if              ; If condition check
