AND

Return the logical and of the the two boolean arguments.

Synopsis

and <bool-exp> <bool-exp>

Arguments

Exactly two arguments are required, and they must evaluate to Booleans.

Returns

Returns the logical and of the two argument expressions.

Example

        make “foo and true true                ; foo = true
        make “foo and true false                ; foo = false
        make “foo false
        make “bar and :foo true                ; bar = false

See Also
