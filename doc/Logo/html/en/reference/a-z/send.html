SEND

Send the argument value to the default serial port. All of the base data types can be sent, including strings.

send also accepts an array argument. This version also accepts a logical serial port number.

All of the base data types can be sent. Note that numerical values are sent as binary data in network byte order. If you want to send a readable string representation then use tostring.

Synopsis

send <value>
send <array-name> <num-items> <port>

Arguments

<value> - a single value to send

<array-name> - name of an array to send

<num-items> - the number of array items to send

<port> - the serial port to send to

    Returns

    Nil.

    Example

            make "foo 43
        send :foo           ; send binary value in network byte order
        send tostring :foo  ; send '42'

          ; Get and send analog data from selected ports
        make "serialPort 1
        array [analogData 5]
        make "next 0
        foreach aport [1 3 4 5 7]
        [
            aset analogData :next analogin :aport
            make "next :next + 1
        ]
        send analogData 5 :serialPort    ; send the array data to the required port

    See Also

    newserial? serial tostring
