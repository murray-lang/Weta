FOR

Execute statements in a block until a counter reaches a given value. The counter's start value, target value, step size and step direction are required

Synopsis

for [ <counter> <from> <to> <step>] [ <statements>* ]

Where

<counter> - the name of the counter variable

<from> - the start value of the counter

<to> - the stop value of the counter

<step> - the increment value for each iteration. Can be negative.

<statements> - zero or more program statements

    Example

                  ; Initialise an array
          array [digits 10]
          for [i 0 9 1]
          [
              aset :digits :i 0
          ]

    See Also

    do.while foreach forever if ifelse loop repeat while
