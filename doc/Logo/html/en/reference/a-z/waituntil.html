WAITUNTIL

Wait until the given expression evaluates to true.

Synopsis

waituntil [ <expression>]

Where

<expression> - code that evaluates to a Boolean

    Example

              waituntil [switch1]     ; Wait for a button to be pressed
          waituntil [newserial?]  ; Wait for new serial data

          make "threshold 45
          waituntil [sensor2 <= :threshold] ; Wait for an analog value to fall
