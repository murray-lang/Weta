RANDOM

VM Stack
BeforeAfter
  
int16 random value

Abstract

Push the a random int16 value onto the stack. The number will be between -32767 and 32767.

Note that this instruction is inherited from the Cricket Logo virtual machine. To provide control over the range of random numbers returned, there is also randomxy.

Synopsis

random

Arguments

Nil.

Returns

Returns a random number between -32767 and 32767 (ie. int16).
