MATH.TRUNC

VM Stack
BeforeAfter
  
valuevalue truncated

Abstract

Pop the value from the top of the stack, and push the value with any fractional part removed (truncated).

The data type expected by math.trunc is determined by the last with<type> instruction. The valid options for math.trunc are: withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling math.trunc is to use one of the prefixed versions of the the instruction, namely:

math.ftrunc or math.dtrunc

The assembler will expand these to a corresponding with<type> instruction followed by math.trunc.

Synopsis

value

math.trunc

Arguments

<value> - the value to truncate

    Note

    This synopsis shows an immediate value being explicitly placed onto the stack. However, the value could be the result peripheral I/O, function calls or any number of mathematical expressions. Or it could be an arbitrary combination of all of these.

    Returns

    Returns the given value with any fractional part removed (truncated).
