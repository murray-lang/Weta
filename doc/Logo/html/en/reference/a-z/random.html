RANDOM

If there is no argument then random returns a pseudo random number between 0 and 32767.

If there are two arguments then random returns a pseudo random number between limits set by the first and second arguments. These arguments are expected to be signed shorts, so the maximum limits are between -32767 and 32767

Synopsis

random
random<lower-limit> <upper-limit>

Arguments

<lower-limit> - the lower limit of the random number range (short)

<upper-limit> - the upper limit of the random number range (short)

    Returns

    If there is no argument then a pseudo random number between 0 and 32767.

    If there are two arguments then a pseudo random number between <lower-limit> and <upper-limit>.

    Example

              make "foo random
          make "bar random 0 255
