MATH.POW

VM Stack
BeforeAfter
  
exponent
basebaseexponent

Abstract

Pop an exponent from the top of the stack then pop the base. Push the value of base to the power of exponent.

The data type expected by math.pow is determined by the last with<type> instruction. The valid options for math.pow are: withfloat and withdouble.

An alternative to explicitly calling one of the with<type> instructions before calling math.pow is to use one of the prefixed versions of the the instruction, namely:

math.fpow or math.dpow

The assembler will expand these to a corresponding with<type> instruction followed by math.pow.

Synopsis

base

exponent

math.pow

Arguments

<base> - the value to be multiplied

<exponent> - the number of times base should be multiplied by itself

    Note

    This synopsis shows immediate values being explicitly placed onto the stack. However, the values could be the result peripheral I/O, function calls or any number of mathematical expressions. Or it could be an arbitrary combination of all of these.

    Returns

    Returns base to the power of exponent.
