GLOBAL

VM Stack
BeforeAfter
  
variable address

Abstract

Push the address of the given global variable onto the stack. Currently the assembler only accepts the label/name of a global variable, which it resolves to an address.

Synopsis

global    <symbol>

Arguments

<symbol> - name of a global variable

Returns

Returns a variable address (by virtue of it being pushed onto the stack)
