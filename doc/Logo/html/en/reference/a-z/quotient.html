QUOTIENT

Return the value that results from dividing the first argument by the second argument. This is the prefix version of the infix / operator. Neither argument is modified in the process.

Synopsis

quotient <dividend> <divisor>

or...

<dividend> / <divisor>

Arguments

<dividend> - the value to be divided

<divisor> - the number of divisions

    Returns

    Return the value that results from dividing <dividend> by <divisor>.

    Example

              make “foo 6
          make “bar quotient :foo 2    ; bar == 3
          make “foo :bar / 3           ; foo == 1
